How It Works

This tool analyzes cryptocurrency risk using traditional finance metrics. Enter one or more crypto assets (using their CoinGecko ID like "bitcoin" or "ethereum") to calculate:

  • Return & Volatility - Annualized return and standard deviation (risk)
  • Maximum Drawdown - Largest peak-to-trough loss during the period
  • Sharpe Ratio - Risk-adjusted returns vs. US Treasury bonds (most common)
  • Sortino Ratio - Risk-adjusted returns focusing only on downside risk
  • Market Correlations - How each asset moves relative to S&P 500 and Bitcoin
